Joe v0.3.0 User's Guide

Node type (joe.gui)

The Node widget is the abstract base class for the JavaFX widget hierarchy. This abstract type provides features available for all widgets.

Methods

  • node.getId() → joe.String
  • node.getStyle() → joe.String
  • node.getStyleClasses() → joe.List
  • node.isDisabled() → joe.Boolean
  • node.isVisible() → joe.Boolean
  • node.setDisable(flag) → this
  • node.setId(id) → this
  • node.setStyle(style, ...) → this
  • node.setVisible(flag) → this

Methods

node.getId()

node.getId() → joe.String

Gets the node's ID string, or null if not defined.

node.getStyle()

node.getStyle() → joe.String

Gets the node's FXCSS style, or null if none. The style is a single CSS string which can contain multiple style settings, separated by semicolons.

node.getStyleClasses()

node.getStyleClasses() → joe.List

Gets the list of the node's FXCSS style class names. Values must be valid CSS style class strings.

node.isDisabled()

node.isDisabled() → joe.Boolean

Returns true if the node has been disabled, and false otherwise.

node.isVisible()

node.isVisible() → joe.Boolean

Returns true if the node is marked "visible", and false otherwise.

Note: if false, the node will occupy its usual space in the window, but will be transparent.

node.setDisable()

node.setDisable(flag) → this

Sets the node's disable flag. If true, this node and its descendants in the scene graph will be disabled.

node.setId()

node.setId(id) → this

Sets the node's ID string.

node.setStyle()

node.setStyle(style, ...) → this

Sets the node's FXCSS style. The caller can pass multiple style strings, which will be joined with semicolons.

node.setVisible()

node.setVisible(flag) → this

Sets the node's visible flag.

Note: if false, the node will occupy its usual space in the window, but will be transparent.
